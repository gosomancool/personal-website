# Personal Website Backend (C++)

This is a personal project aimed at building a lightweight, high-performance web server backend using C++. 

## Purpose
I am developing this project to host my personal blog and portfolio. I plan to use the **.us.kg** domain to provide a professional entry point for my site.

## Features
- Lightweight HTTP request handling
- High-performance C++ implementation
- Future integration with a SQL database for blog posts

## How to Build
```bash
g++ main.cpp -o server
./server
